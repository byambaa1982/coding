#### Lesson 2: Variables and Data Types (45 min)
**Content Type**: Video + Interactive Exercises

**Key Topics**:
- What are variables? (Think of them as labeled boxes)
- Creating and naming variables (naming conventions)
- Basic data types: strings, integers, floats, booleans
- Type conversion and type checking
- Common mistakes and how to avoid them

**Learning Outcomes**:
- Create variables with meaningful names
- Identify different data types
- Convert between data types
- Debug simple type-related errors

**Interactive Exercises** (5 exercises):
1. Create variables to store personal information (name, age, height)
2. Convert temperature from Fahrenheit to Celsius
3. Calculate area of a rectangle using variables
4. Fix the data type errors in provided code
5. Create a simple calculator that adds two numbers

**Test Cases**:
- Variable assignment validation
- Type checking assertions
- Output format verification
- Edge case handling (negative numbers, decimals)

#### Lesson 3: Working with Strings (45 min)
**Content Type**: Video + Interactive Exercises

**Key Topics**:
- Creating strings (single vs double quotes)
- String concatenation and repetition
- String methods (upper, lower, strip, replace)
- String formatting (f-strings)
- Multi-line strings
- Escape characters

**Learning Outcomes**:
- Manipulate strings using built-in methods
- Format strings for output
- Handle user input text

**Interactive Exercises** (6 exercises):
1. Create a personalized greeting message
2. Format a full name from first and last name
3. Count characters in a sentence
4. Convert text to title case
5. Replace words in a sentence
6. Build a simple Mad Libs game

### **Section 2: Control Flow and Logic** (3-4 hours)

#### Lesson 4: Making Decisions with If Statements (60 min)
**Content Type**: Video + Interactive Exercises + Quiz

**Key Topics**:
- Boolean logic (True/False)
- Comparison operators (==, !=, >, <, >=, <=)
- If, elif, else statements
- Nested conditionals
- Logical operators (and, or, not)
- Common patterns and best practices

**Learning Outcomes**:
- Write conditional statements
- Compare values and make decisions
- Combine multiple conditions
- Avoid common logic errors

**Interactive Exercises** (7 exercises):
1. Check if a number is positive, negative, or zero
2. Determine if a person is eligible to vote
3. Grade calculator (A, B, C, D, F based on score)
4. Simple password validator
5. Leap year checker
6. Temperature advisory system
7. Login authentication simulator

**Quiz**: 10 multiple-choice questions on conditionals and logic

#### Lesson 5: Loops - Repeating Actions (60 min)
**Content Type**: Video + Interactive Exercises

**Key Topics**:
- Why use loops? (Don't Repeat Yourself principle)
- While loops (condition-based repetition)
- For loops (iterating through sequences)
- Range function
- Break and continue statements
- Nested loops
- Loop patterns (counting, accumulating, searching)

**Learning Outcomes**:
- Choose appropriate loop type for different scenarios
- Iterate through sequences
- Control loop execution with break/continue
- Avoid infinite loops

**Interactive Exercises** (8 exercises):
1. Print numbers from 1 to 100
2. Calculate the sum of numbers from 1 to N
3. Multiplication table generator
4. Find all even numbers in a range
5. Countdown timer simulation
6. Password retry system (3 attempts)
7. Pattern printing (stars, pyramids)
8. FizzBuzz challenge

**Content Type**: Video + Interactive Exercises

**Key Topics**:

- Accessing list items (indexing, slicing)
- Modifying lists (append, insert, remove, pop)

- Create and manipulate lists
- Access elements using indices
- Perform common list operations
- Iterate through lists with loops

**Interactive Exercises** (8 exercises):

**Key Topics**:

- Accessing values by keys
- Adding, modifying, and deleting items

**Learning Outcomes**:

**Interactive Exercises** (6 exercises):
